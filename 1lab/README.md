## Описание проекта

Данный проект представляет собой веб-приложение для демонстрации работы генетического алгоритма при решении задачи поиска кратчайшего пути в графе. Проект реализован полностью на JavaScript и работает на клиентской стороне.

## Теоретические основы

### Генетические алгоритмы как метод вычислительного интеллекта

Генетические алгоритмы относятся к методам **вычислительного интеллекта** (computational intelligence) – направлению искусственного интеллекта, основанному на:

- Извлечении закономерностей из данных
- Моделировании естественных процессов, связанных с интеллектуальным поведением
- Использовании числового представления исходных данных

### Основные свойства методов вычислительного интеллекта

1. **Адаптивность** – способность системы изменять своё поведение в зависимости от изменений окружающей среды
2. **Самоорганизация** – способность системы самостоятельно формировать структуру без внешнего управления
3. **Способность обобщения** – умение извлекать общие закономерности из конкретных примеров
4. **Биологически инспирированная сущность** – использование принципов, заимствованных из живой природы

### Подход к созданию интеллектуальных систем

Генетические алгоритмы относятся к **системам, которые действуют рационально**, так как они направлены на поиск оптимального решения задачи путём максимизации функции приспособленности (fitness function).

## Функционал

### Основные возможности

1. **Создание и настройка сети**
   - Настройка размера сети (4-16 узлов)
   - Выбор отправителя и получателя
   - Генерация случайной сети
   - Визуализация матрицы смежности

2. **Настройка генетического алгоритма**
   - Размер популяции (10-200)
   - Максимальное количество поколений (10-1000)
   - Вероятность мутации (0.01-0.5)
   - Вероятность скрещивания (0.1-1.0)
   - Размер турнира (2-10)
   - Методы селекции: турнирная, рулеточная, ранговое
   - Методы скрещивания: однородное, одноточечное, двухточечное

3. **Режимы работы**
   - **Циклический**: автоматическое выполнение до завершения
   - **Пошаговый**: ручное управление каждым шагом эволюции

4. **Визуализация**
   - Отображение найденного пути
   - Сравнение с оптимальным решением (алгоритм Дейкстры)
   - Графики сходимости (Chart.js)
   - Отображение популяции в пошаговом режиме

5. **Эксперименты**
   - Автоматическая серия из 6 экспериментов
   - Анализ влияния параметров
   - Сводка результатов

## Технические детали

### Представление решения

- **Хромосома**: последовательность узлов сети, образующих путь от отправителя к получателю
- **Ген**: узел в графе
- **Функция приспособленности**: сумма весов рёбер, входящих в путь (чем меньше, тем лучше)

### Операторы генетического алгоритма

1. **Селекция**:
   - Турнирная
   - Рулеточная
   - Ранговая

2. **Скрещивание**:
   - Однородное
   - Одноточечное
   - Двухточечное

3. **Мутация**: случайная замена промежуточного узла пути

### Сравнение с оптимальным решением

Для оценки качества работы алгоритма используется алгоритм **Дейкстры**, который находит точное оптимальное решение задачи поиска кратчайшего пути.

## История развития ИИ

Проект демонстрирует применение методов вычислительного интеллекта, которые получили широкое распространение начиная с 1980-х годов:

- **1943 г.** – работа Мак-Калока и Питтса о математической модели искусственного нейрона
- **1956 г.** – термин "искусственный интеллект" предложен Джоном Маккарти
- **1980-е** – коммерческое применение экспертных систем
- **1986 г.** – возрождение интереса к искусственным нейронным сетям
- **Конец 1980-х – настоящее время** – зрелый этап развития, методы вычислительного интеллекта
